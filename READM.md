
# autops介绍
autops是基于Python3开发的一款CMDB+IT审计堡垒机，支持web terminal和终端ssh。

python3 manage.py makemigrations
python3 manage.py migrate
python3 manage.py createsuperuser
python manage.py runserver


yum -y install dmidecode
